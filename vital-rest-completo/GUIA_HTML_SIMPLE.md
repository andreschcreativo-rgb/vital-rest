## PASO 2: DESCOMPRIMIR

1. Abre tu **Explorador de archivos**
2. Ve a **Descargas**
3. Busca **`vital-rest-html.zip`**
4. Haz **clic derecho** → **"Extraer todo..."**
5. Extrae a: **`Desktop\vital-rest`** (crea una carpeta nueva)

**Resultado:** Tienes una carpeta con un archivo `index.html` adentro

## PASO 4: SUBIR A GITHUB

### 4A: Abre terminal en esa carpeta

1. Abre **Símbolo del sistema** (Windows + R → `cmd`)
2. Escribe:
   ```
   cd Desktop\vital-rest
   ```
3. Presiona Enter

### 4B: Inicializa Git

Escribe:
```
git init
git add .
git commit -m "Vital Rest landing page"
```

Presiona Enter en cada línea.

### 4C: Crea el repo en GitHub

1. Ve a: https://github.com/new
2. Nombre: `vital-rest` (cualquier nombre)
3. Click "Create repository"

### 4D: Sube el código

En la terminal, escribe:

```
git remote add origin https://github.com/TUUSUARIO/vital-rest.git
git branch -M main
git push -u origin main
```

*(Cambia `TUUSUARIO` por tu usuario de GitHub: `andreschcreativo-rgb`)*

Presiona Enter y espera.

---

## PASO 5: DEPLOYAR EN VERCEL (EL MÁS IMPORTANTE)

### 5A: Ve a Vercel

1. Abre: https://vercel.com
2. Login con tu cuenta
3. Click "Add New..." → "Project"

### 5B: Importa GitHub

1. Selecciona tu repo `vital-rest`
2. Click "Import"

### 5C: Configuración

**Importante:** En la pantalla de deployment, busca:

**Root Directory:** Cambia a `.` (punto)

*(Porque el `index.html` está en la raíz, no en una subcarpeta)*

### 5D: Deploy

Click "Deploy"

**¡LISTO!** En 30 segundos tu página está en vivo 🎉

Verás algo como:
```
https://vital-rest.vercel.app
```
